# OrangeHRM-Selenium-Automation
Selenium automation project for OrangeHRM covering all UI elements.
OrangeHRM Selenium Automation Project

Tools Used:
- Java
- Selenium WebDriver
- TestNG
- Maven

Features Automated:
- Login
- Dashboard navigation
- Add Employee
- Employee List table validation
- Logout
